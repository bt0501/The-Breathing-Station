# The-Breathing-Station
"Интерактивная станция для крипто-трейдинга с живым пульсом рынка, визуализацией торговых сессий и аудио-алертами волатильности."
# 🚀 Crypto Station v8.4

Интерактивная панель для мониторинга криптовалютного рынка в реальном времени.

### 🛠 Основные функции:
* **Live BTC/USDT:** Обновление цены через WebSocket Binance.
* **Таймлайн сессий:** Визуализация торговых сессий (Сидней, Токио, Лондон, Нью-Йорк) с автоматическим расчетом UTC.
* **Рыночный пульс:** Индекс страха и жадности, цена газа ETH и доминация BTC.
* **Аудио-визуальное "дыхание":** Звуковые алерты при резких скачках волатильности.
* **Интеграция TradingView:** 15-минутный график с техническими индикаторами.

### 🖥 Технологии:
* HTML5 / CSS3 (Grid & Flexbox)
* JavaScript (ES6+)
* WebSockets API
* Google Translate API (для новостей)
